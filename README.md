# 🎓 Student Performance Analyzer

A beginner-friendly data analysis project built using **Python**, **Pandas**, and **NumPy** to analyze student performance across multiple subjects.

## 📌 Project Overview

This project stores marks of students in different subjects and performs various analyses such as calculating average marks, finding the topper and lowest scorer, assigning grades automatically, and generating statistical insights.

The project was created to practice fundamental concepts of **Pandas** and **NumPy** through a real-world dataset.

---

## 🚀 Features

- Store marks of students in multiple subjects.
- Calculate average marks for each student.
- Identify the topper and lowest scorer.
- Automatically assign grades based on average marks.
- Calculate:
  - Mean
  - Median
  - Standard Deviation
  - Percentiles
- Filter students based on performance.
- Sort students according to their average marks.
- Analyze overall class statistics.

- Python
